# Meridian

> **A modern, keyboard-first web console interface designed for speed and efficiency.**

Meridian is a beautiful web application UI tailored for managing logs, metrics, and traces. Designed with a focus on a "keyboard-first" experience, it helps teams managing production environments to significantly reduce their Mean Time To Resolve (MTTR) incidents.

## ✨ Features

- **⌨️ Keyboard-First Interface:** Navigate and resolve issues at lightning speed without taking your hands off the keyboard.
- **📊 Unified Console:** A single pane of glass for all your logs, metrics, and traces.
- **🎨 Modern & Responsive UI:** Built with clean HTML, CSS, and styled with Font Awesome icons.
- **⚡ Performance Focused:** Lightweight and fast, giving you the information you need instantly.

## 🛠️ Technologies Used

- **HTML5** - For structural semantic markup.
- **CSS3** - Custom styling, flexbox/grid layouts, and responsive design.
- **Font Awesome** - For scalable vector icons.

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

You only need a modern web browser (Chrome, Firefox, Safari, Edge) to view this project. No server setup is required.

### Installation

1. **Clone the repository** (or download the ZIP file):
   ```bash
   git clone <repository-url>
   ```
2. **Navigate to the project directory:**
   ```bash
   cd Meridian
   ```
3. **Open the application:**
   Simply double-click the `Index.html` file, or open it directly in your web browser.
